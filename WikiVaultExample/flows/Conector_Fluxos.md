# Conector: Fluxos do Sistema

Este documento atua como um índice para os comportamentos ponta-a-ponta (end-to-end) da aplicação grapy-wiki.

- [[renderizacao-nota]] — Como uma nota sai do disco cru e vira HTML protegido na tela.
- [[carregamento-grafo]] — A mágica por trás do mapa de conhecimento inicial e a física de JS puro.
- [[sync-webhook]] — Como manter a wiki atualizada automaticamente após pushes no GitHub sem downtime.
- [[resolucao-wikilinks]] — O mecanismo de busca reversa que encontra as notas apenas pelo nome no meio do texto.
- [[rbac-pastas]] — O fluxo de segurança que blinda acesso a notas e anexos baseado estritamente no disco local.
